# Bhalbarathi RAG System without LLM

## Overview
The Bhalbarathi RAG (Retrieval-Augmented Generation) system is a novel approach to enhancing information retrieval processes without the use of Large Language Models (LLMs). This project aims to combine traditional retrieval techniques with modern generation methods to improve accuracy and efficiency in generating relevant responses.

## Features
- **Efficient Retrieval**: Leverages classical algorithms for information retrieval to quickly fetch pertinent data.
- **Augmentation Mechanism**: Integrates generation techniques that enhance the context and relevance of the retrieved information.
- **User-Friendly Interface**: Provides an intuitive interface for users to interact with the system and obtain results effortlessly.
- **Customizable**: Designed to allow developers to adjust parameters and components according to specific requirements.
